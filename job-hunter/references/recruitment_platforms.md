## 各平台高效搜索技巧

### 1. BOSS直聘

**URL 构建模式：**
```
基础URL: https://www.zhipin.com/c101210100-p100101/
筛选参数: 
  - district=110106 (滨江区)
  - query=Java
  - experience=103 (3-5年) | 104 (5-10年)
完整示例: https://www.zhipin.com/c101210100-p100101/?district=110106&query=Java&experience=104
```

**搜索流程：**
1. 用 WebSearch 搜 `{公司名} {岗位} {城市} BOSS直聘`
2. 提取公司名、薪资范围、经验要求、具体岗位标题、发布时间
3. 如需详情，用 WebFetch 打开具体岗位链接

**关键筛选维度：**
- 城市编码: c101210100 = 杭州
- 滨江区 district: 110106
- 经验: 101=应届, 102=1-3年, 103=3-5年, 104=5-10年, 105=10年以上
- 学历: 202=大专, 203=本科, 204=硕士

### 2. 猎聘

**URL 构建模式：**
```
基础: https://www.liepin.com/city-hz/zp{关键词拼音}/
示例: https://www.liepin.com/city-hz/zpjavakaifagongchengshi/
```

**搜索流程：**
1. 用 WebSearch 搜 `{公司名} {岗位} {城市} 猎聘`
2. 猎聘搜索结果偏中高端，注意筛选薪资 20k+ 的岗位
3. 猎聘需要注册才能看完整JD，可通过 WebSearch 绕过限制

### 3. 智联招聘

**特点：**
- 外企德科、华为OD等外包岗位多在智联
- 传统行业（银行、保险、制造）IT岗位丰富
- 搜索策略: `{公司名} {岗位} {城市} 智联招聘`

### 4. 51job / 前程无忧

**特点：**
- 岗位基数大、质量参差不齐，适合海投
- 外包公司（中软、软通、文思海辉等）岗位密集
- 搜索策略: `{公司名} {岗位} 51job {城市}`

## 时效过滤策略（强制执行）

**仅展示近3个月（90天）发布或更新的岗位。**

### 各平台时间识别特征

| 平台 | 时间展示方式 | 判断方法 |
|------|------------|----------|
| BOSS直聘 | "刚刚活跃""X天前发布""X周前更新" | ≤90天即纳入 |
| 猎聘 | "X天前""X小时前""发布于 YYYY-MM-DD" | 日期 ≤90天即纳入 |
| 智联招聘 | "发布于XX天前""更新于XX天前" | ≤90天即纳入 |
| 51job | "更新日期: YYYY-MM-DD" | 直接比较日期 |
| 公司官网 | 通常无明确日期 | 标注 `—`，按有效处理 |

### 搜索时的时间提示

WebSearch 搜索时在 query 中增加时间限定词：
- `{公司名} {岗位} {城市} 招聘 2026` — 今年的招聘
- `{公司名} 社招 {岗位} 2026年6月` — 近月岗位
- 搜索时关注结果中是否出现"半年前""发布于2025""2024年"等过期标记

## 提取直达链接策略（强制执行）

**目标：每条岗位记录必须链接到具体岗位详情页，而非列表页或首页。**

### 各平台详情页 URL 特征

| 平台 | 详情页 URL 特征 | 示例 |
|------|----------------|------|
| BOSS直聘 | `/job_detail/` + 16位hex | `zhipin.com/job_detail/xxx.html` |
| 猎聘 | `/job/` + 数字ID + `.shtml` | `liepin.com/job/1976138447.shtml` |
| 51job | `/all/` + 数字ID + `.html` | `jobs.51job.com/all/171363082.html` |
| 智联招聘 | `/jobdetail/CC` + 随机串 | `zhaopin.com/jobdetail/CCxxx.htm` |
| 牛客网 | `/job/detail/` + 数字 | `nowcoder.com/job/detail/xxx` |

### 提取流程

1. 从 WebSearch 返回的搜索结果中，筛选 URL 包含上述特征的链接
2. 优先取结果的 **URL 字段**，这是可点击直达的
3. 对确认匹配的详情页链接，用 WebFetch 验证相关性
4. 标题或职责中有一个维度的关键词匹配即视为有效岗位

### 无法提取详情页时的降级处理

当确有公司/岗位但无法获取详情页 URL 时：
- 输出格式：标注 `[在招聘页搜索]` + 附公司招聘首页链接
- 备注列写明搜索关键词（如"进入后搜索 Java 滨江"）

## 来源标注规则

每条岗位的「来源」列按以下规则填写：

| 链接域名 | 来源列标注 |
|----------|-----------|
| 公司自有域名（hikvision.com, hundsun.com 等） | `⭐官网` （若平台上也搜不到该岗位，加注「官网独家」） |
| zhipin.com | `BOSS直聘` |
| liepin.com | `猎聘` |
| zhaopin.com | `智联招聘` |
| 51job.com | `51job` |
| hhrc.com.cn | `高新人才网` |
| nowcoder.com | `牛客网` |

**官网独家判定**：某公司某岗位在官网招聘页存在，但在 BOSS直聘/猎聘/智联 三大平台均搜索不到 → 来源标 `⭐官网独家`

```markdown
## {城市}{区} {岗位} 社招岗位汇总

> 📍 基准地址：{地址} | 🕐 数据采集：{日期} | ⏳ 仅展示近3个月在招岗位

| 序号 | 公司 | 距离 | 岗位 | 薪资 | 经验 | 更新时间 | 来源 | 投递链接 |
|:---:|------|------|------|------|------|----------|------|----------|
| 1 | XX公司 | ~500m | Java开发 | 15-25k | 3-5年 | 2026-06 | ⭐官网 | [投递](url) |
|   |      |       | 架构师 | 30-45k | 5年+ | 2026-05 | ⭐官网 | [投递](url) |
| 2 | YY公司 | ~1.2km | Java后端 | 20-30k | 3-5年 | 2026-06 | BOSS直聘 | [投递](url) |

> ⭐官网 = 仅在官网招聘，平台未收录；来源标注对应平台名称。
```

## 注意事项

1. **时效性**：招聘信息变化快，仅展示近3个月岗位，标注数据采集时间
2. **去重**：同一岗位可能出现在多个平台，以官网/直投链接优先
3. **外企德科/HW OD**：这类是外包岗，标注清楚「外包」性质
4. **小公司**：部分公司无独立官网招聘页（如2023年后成立的小微企业），如实说明
5. **薪资**：各平台薪资口径不一（含/不含年终、税前/税后），标注时注明平台来源
6. **链接有效性**：输出前确认每个链接指向具体的岗位详情页，禁止只给招聘首页或搜索列表页
7. **匹配宽松原则**：标题或岗位职责中有一个维度匹配目标岗位关键词即可纳入结果
8. **距离优先**：同区域内按距离近→远排列
